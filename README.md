# LLM Cost Estimates: Web UIs vs APIs

## Purpose Statement

A few *back of the envelope* style cost estimates to compare the cost of using large language models (LLMs) via web UIs (e.g. ChatGPT) and using them through APIs (directly or with third-party frontends like, for instance, Jan).
